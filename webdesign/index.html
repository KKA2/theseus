About

Theseus is designed to allow for a greater ease of access in navigating the library to find a book. To use, simply enter a book code and you will be directed to the specific floor and shelf in which the book is located.

Courtesy of the 2016 Hesburgh Hack-A-Thon
Using the maze solver algorithm by scharissis
Designed with Bootstrap, Smooth Scroll and Font Awesome

Creators: Kate Barlock, Katricia Herring, and Ann Keenan
